# TrashMap - Roadmap

## Vision
Enable communities worldwide to map, track, and collectively clean up illegal dumping sites through open-source technology.

## Version 1.0 - MVP (Current)
### Features
- [x] Interactive map with MapLibre GL JS
- [x] Report incidences (location, type, quantity, description)
- [x] List all incidences
- [x] View incidence details
- [x] Update incidence
- [x] Delete incidence
- [x] Spatial queries (nearby incidences)
- [x] Statistics dashboard
- [x] PostGIS integration
- [x] Seed data (50 Venezuela reports)
- [x] Docker deployment
- [x] Railway deployment ready
